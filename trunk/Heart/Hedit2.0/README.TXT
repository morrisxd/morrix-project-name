SUMMARY

Hexadecimal editor for binary files. Unlimited file size (easily works 
with GB-sized files), insert and delete, binary and text search, 
selectable fonts, clipboard support, multiple document interface. 
Integrates with the Windows 95 or NT 4.0 shell.

INSTALLATION

1. Extract all the HEdit files into a directory on your hard disk.
2. Start HEDIT.EXE.
3. Now you can open files with HEdit by using the right mouse button in 
the shell.


REQUIREMENTS

Requires Windows 95 or Windows NT running on Intel platform.
